# RoadSafe

## Startup
From the project root, run:

```bash
cd repo
docker compose up --build
```

## Credentials
Default administrator account:
- Username: `admin`
- Password: `Admin@123456`

## Services
- Frontend: `http://localhost:5173`
- Backend: `https://localhost:4000`

## Verification
Run the full automated test suite from `repo/`:

```bash
./run_tests.sh
```
